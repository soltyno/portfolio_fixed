01

BASIC WEB

In this theme I was introduced to the basic terminology within user interface design, digital communication, user testing, and responsive web design. I have refreshed my skills in html and css coding as well as basics in Photoshop and Illustrator.

02

BASIC CONTENT

This module included development of video production, as a dramaturgy elements, film language, recording and editing techniques. I was expected to work in several different teams during this theme, so I have gained some experience working in the group as well.

03

BASIC ANIMATION

This theme contained both individual and group productions. The main goal was to create a few interactive animations with an interactive story with focus on the story, a style, character design, animations and interactivity designed for a specific target group. I had to use my skills in sketching, hand drawing and then drawing visual universe and characters in illustrator. I have learned about animation principles which I used later and I had to expand my skills in HTML and CSS as well, because I had to add SVG files and audio elements to the site as well as use advanced selectors, transitions, keyframe animations, querySelector, events & eventListeners and so on. Despite the fact that it was a bit difficult I really enjoyed this theme.

04

BASIC UX

This fourth theme gave me a basic understanding of how interaction between users and user interfaces works, as well as which theories, tools and methods I can use for research, design, testing and production of digital solutions for smartphones so that my work is based on specific insights about actual users instead of my own feelings.

Basic UX

The fourth theme was mostly about group work. This mean that everyone has the same rights except a decider. The idea of the whole theme was remaking and improving the fronter with the aim to satisfy users. We had to start thinking like users of the fronter and also interviewing other users. Besides that we did also research, sketching, testing prototypes and deciding in a group. This whole theme gave me plenty of experience in UX and I believe that I can use these experience later in next projects.

Design sprint

As a group we had so many thoughts about improving the fronter, however, we could choose only one user journey, and we had little time as well, so we couldn't apply all our ideas. Design sprint was new to me and I liked the idea that everyone has the same right. However, our design sprint was a bit adjusted in the meaning that it looked more friendly and unprofessional, and we didn't use stickers for voting for everything, but I think it's normal as we were only five students doing it for the first time. I think that we did well in this assignment and everyone from a group is satisfied about the results.

Native app

Our last project was to create Native app of the Fronter. As a new group we spent last week by doing Design Sprint, so researching, brainstorming, putting our ideas on the wall, deciding, sketching, interviewing and after all prototyping the final app in XD. Design Sprint was without any problems as we were already familiar with it and prototyping was easy for us as well. However, in this assignment, we learn how to use UI kits in XD, so our prototype would look more realistic for users of iOS or Android devices.

Summary

If I should sum up the whole semester in brief, I would say that I’m absolutely delighted that I choose this programme because it’s exactly what I wanted and at the same time a lot more that I expected. Of course, there are some things and situations that I don’t like so much but these bright sides significantly outweigh dark sides. I liked the whole content of the 1.
